# College-Management-Dashboard-PowerBI
A clean and interactive College Management dashboard built using Power BI. Includes insights on students, departments, attendance, fees, and performance KPIs.

Key Features
a)Student count by department  
b)Gender distribution  
c)Attendance overview  
d)Department-wise insights  
e)KPI cards for quick decision-making  
f)Clean and minimal UI design

Tools Used
- Power BI
- Excel (Data source)

Project Files
Dashboard (PDF version + .pbix)

About This Project
This dashboard was created to practice data visualization, KPI building, and creating a clean, structured report UI for academic management systems.
